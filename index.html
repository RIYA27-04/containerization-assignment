📌 Introduction

Containerization is a lightweight virtualization technology that allows applications to run in isolated environments using Docker.

🐳 Docker Commands

docker run hello-world
docker pull ubuntu
docker run -it ubuntu
docker ps
docker ps -a
docker stop <container_id>
docker rm <container_id>

RUN apt update
